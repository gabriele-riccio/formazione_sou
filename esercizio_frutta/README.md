# Esercizio: Gestire file .csv con AWK, Grep e Sed

## Obiettivo
Dato un file `.csv`, cercare la stringa "banana". Se viene trovata una corrispondenza nella riga, il programma deve stampare esclusivamente il **terzo campo**.

## File inclusi
- `file_frutta.csv`: File contenente dati fittizi per testare gli script.
- `file_frutta.awk`: Soluzione in AWK.
- `file_frutta_grep.sh`: Soluzione usando Grep.
- `file_frutta_sed.sh`: Soluzione in Sed.
